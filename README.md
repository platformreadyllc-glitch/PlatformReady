# PlatformReady

TypeScript/NestJS backend for PlatformReady.

## Getting started

```bash
cd backend
npm install
npm run start:dev
```

## Scripts

| Command | Description |
|---|---|
| `npm run start:dev` | Start in watch mode |
| `npm run build` | Compile TypeScript |
| `npm run lint` | Run ESLint |
| `npm test` | Run Jest tests |

## Branch rules

- All PRs targeting `main` must pass CI before merge
- Direct commits to `main` are not allowed
